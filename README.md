Учебный проект по автотестированию API сервиса **Яндекс Самокат**

## Структура проекта

Sprint_7/
├── helpers/              # вспомогательные функции (генерация данных, регистрация курьера) 
│   ├── test_create_courier.py # интерфейс для работы с API (эндпоинты и методы)    
│   ├── test_login_courier.py # генерация данных и регистрация курьера
├── tests/                # тесты для каждой ручки 
│   ├── test_create_courier.py # тесты создания курьера
│   ├── test_login_courier.py # тесты авторизации курьера
│   ├── test_create_order.py # тесты создания заказа
│   └── test_get_orders.py # тесты получения списка заказов
├── conftest.py           # фикстуры pytest (создание и удаление тестовых данных) 
├── requirements.txt      # зависимости проекта 
└── allure-results/        # Allure-отчет

## Покрытие тестами

    1. Создание курьера: позитивные и негативные сценарии
    2. Логин курьера: позитивные и негативные сценарии (неверный пароль, отсутствие полей, несуществующий пользователь)
    3. Создание заказа: параметризация по цветам (BLACK, GREY, оба цвета, без цвета)
    4. Получение списка заказов: проверка, что возвращается список orders

## Баги
    Ожидаемый результат: согласно документации, при попытке авторизации курьера без обязательного поля password сервер должен возвращать 400 Bad Request
    Фактический результат: сервер возвращает 504 Gateway Timeout
